# Project README

## 1. Platform Used
- **Windows 10**

## 2. Software / Tools Used
- **Node.js**
- **Next.js**
- **React**

## 3. Commands to Run the Code

### Install all required dependencies
```
npm install
```

### Run the development server (Next.js app)
```
npm run dev
```

### Run the Node server (if your project contains server.js)
```
npm start
```
